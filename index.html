Description: Tagger corpus.

Example sentence: Google;American;technology;company;software Google is an American multinational technology company specializing in Internet-related services and products that include online advertising technologies, search, cloud computing, and software.

Every line starts with target entities of which the number is without limitation. Target entities and sentences need to be split by "\t" (Tab key)as shown above. It also highlight multiple words in text. Firstly, you need to input the Number of target entities (the default is 2). Secondly, you need to input the number of classes you expect (the default is 2). So far, it supports up to 10 classes. The index of sentences is no need to input. Finally, Paste your sentences here, and click the button upload. You will know how it works.

If you want to save you classification results, you need to click the button "class_i" to download the data that is classified by you by chrome. According to the file name, you will know how to process them.

功能描述：快速语料标注的工具。Moll Tagger中文版。

例句: 谷歌;美国;互联网;云计算、广告 Google（中文名：谷歌），是一家美国的跨国科技企业，致力于互联网搜索、云计算、广告技术等领域，开发并提供大量基于互联网的产品与服务，其主要利润来自于AdWords等广告服务。Google由当时在斯坦福大学攻读理工博士的拉里·佩奇和谢尔盖·布卢姆共同创建，因此两人也被称为“Google Guys”。

每一行以目标实体开始，目标实体的数量不限。目标实体和句子以退格键隔开(Tab键)。 它同时也在文本中高亮显示索要搜索的关键词。首先，你需要输入目标实体的数量（默认是2），然后你需要输入你预期的类别数量（默认是2），目前最多支持10类。最后，将你想处理的句子粘贴进文本框，然后点击上传按钮。

如果你想要保存分类的结果，你需要点击"class_i"按钮下载你分类的数据，推荐使用chrome浏览器下载更为方便。根据下载的文件名称，你将知道如何进行下一步处理。
