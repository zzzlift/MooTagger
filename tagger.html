Tutorial

Example sentence: Google;American;technology company;software Google is an American multinational technology company specializing in Internet-related services and products that include online advertising technologies, search, cloud computing, and software.

Every line starts with target entities of which the number is without limitation. Target entities and sentences need to be split by ";" (; key)as shown above. Firstly, you need to input the Number of target entities (the default is 2). Secondly, you need to input the number of classes you expect (the default is 2). So far, it supports up to 10 classes. The index of sentences is no need to input. Finally, Paste your sentences here, and click the button upload. You will know how it works.

If you want to save you classification results, you need to click the button "class_i" to download the data that is classified by you by chrome. According to the file name, you will know how to process them.
